e = 0

Cerchio

Luogo dei punti equidistanti (distanza r) da un centro O.

Equazioni

x² + y² = r²
(x−a)² + (y−b)² = r²  centro O=(a,b)

Proprietà

  • Eccentricità e = 0 — caso limite ellisse con a = b
  • Simmetria rispetto a infiniti diametri
  • Fuoco unico coincidente col centro
  • Circonferenza C = 2πr; area A = πr²
e = 1

Parabola

Equidistante da un fuoco F e da una direttrice d.

Equazioni

y² = 4px (asse orizzontale)
x² = 4py (asse verticale)

Proprietà

  • Fuoco F=(p,0); direttrice x=−p
  • Vertice V=(0,0); asse di simmetria: asse x
  • Latus rectum = 4p (corda per F ⊥ all'asse)
  • Proprietà riflessiva: raggi ∥ asse → convergono in F
0 < e < 1

Ellisse

PF₁ + PF₂ = 2a per ogni punto P.

Equazione canonica

x²/a² + y²/b² = 1  (a > b > 0)

Proprietà

  • Fuochi F₁,₂=(±c,0), c²=a²−b²
  • Eccentricità e=c/a ∈ (0,1)
  • Direttrici x=±a/e; due assi di simmetria
  • Area A=πab; se a=b → cerchio
e > 1

Iperbole

|PF₁ − PF₂| = 2a per ogni punto P.

Equazione canonica

x²/a² − y²/b² = 1

Proprietà

  • Fuochi F₁,₂=(±c,0), c²=a²+b²
  • Eccentricità e=c/a > 1
  • Asintoti y=±(b/a)x
  • Due rami, due assi di simmetria
  • Iperbole equilatera: a=b, asintoti ⊥

Origini storiche

Intersezioni di un piano con un cono doppio — il tipo dipende dall'inclinazione.

Piano ⊥ asse

Cerchio

Taglia una falda uniformemente.

Piano ∥ generatrice

Parabola

Interseca una falda in curva aperta.

Meno inclinato

Ellisse

Una falda, curva chiusa.

Più inclinato

Iperbole

Entrambe le falde — due rami.

Storia

Apollonio di Perga (III–II sec. a.C.) scrisse gli 8 libri delle "Coniche". Keplero scoprì le orbite ellittiche dei pianeti; Galileo provò che le traiettorie dei proiettili sono parabole; Newton unificò tutto nella gravitazione universale.

Condividere su Google Classroom

Istruzioni per pubblicare questa pagina come materiale didattico per i tuoi studenti.

👩‍🏫 SOLO DOCENTE  Metodo 1 — Pubblica il file HTML

  1. Nell'artifact qui sopra clicca l'icona Scarica (o copia tutto il codice) e salva il file come coniche.html sul tuo computer.
  2. Vai su Google Drive e carica il file (+ Nuovo → Carica file).
  3. Clicca con il tasto destro sul file → Condividi → imposta "Chiunque abbia il link può visualizzare" e copia il link.
  4. Apri Google Classroom → seleziona il corso → Lavori del corso → + Crea → Materiale.
  5. Nel campo link incolla l'URL di Drive. Aggiungi titolo (es. "Sezioni Coniche — pagina interattiva") e istruzioni per gli studenti.
  6. Clicca Pubblica. Gli studenti vedranno il link nella bacheca dei lavori.
⚠️ Google Drive non esegue HTML direttamente nel browser — gli studenti dovranno scaricare il file e aprirlo localmente, oppure usa il Metodo 2.

👩‍🏫 SOLO DOCENTE  Metodo 2 — Pubblica su GitHub Pages (consigliato)

  1. Crea un account gratuito su github.com se non ce l'hai.
  2. Clicca + New repository → nome: coniche → spunta Public → Create repository.
  3. Nella pagina del repository clicca Add file → Upload files e carica coniche.html. Rinominalo index.html prima di caricare.
  4. Vai in Settings → Pages → sorgente: Deploy from a branch → main → / (root) → Save.
  5. Dopo 1–2 minuti la pagina è live su https://tuonome.github.io/coniche/. Copia questo URL.
  6. In Google Classroom → Materiale → + Aggiungi link → incolla l'URL di GitHub Pages → Pubblica.
✅ Con GitHub Pages la pagina si apre direttamente nel browser degli studenti — nessun download necessario. È la soluzione più fluida.
